# Tema

1. Fă-ți propriul fork. Doar din proiectul principal. **O singură dată**. Fork-ul trebuie făcut pe contul tău de GitHub.
2. Clonează fork-ul tău pe PC-ul (inclusiv Mac) propriu
3. Crează fișierele de care ai nevoie pentru implementa designul de [aici](https://www.figma.com/file/xEktFZL9eoqJ7lnmie6ufk/Teme-Front-End-by-Cornel).
4. Fă un commit 
5. Dacă nu reușești să rezolvi complet, poți face commit la ce ai, și vei cere ajutor.
6. Fă push pe GitHub
7. Crează un Pull Request la [proiectul principal](https://github.com/ITSchool-Web-Force/Tema-Grid/pulls)
